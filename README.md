# 🚀 Futuristic To-Do – Modern Task Management App

A visually modern **task management application** built using **React (Vite) and Tailwind CSS v4**, focused on glassmorphism design, smooth interactions, and responsive layout.

---

## 📌 Project Overview

Task management applications require simplicity, clarity, and smooth user experience.

**Futuristic To-Do** is a modern frontend application designed with a premium glass-style interface.  
It combines minimal business logic with a visually engaging futuristic UI to enhance productivity and user experience.

---

## 🎯 Objective

To design and develop a **modern React-based to-do application** that:

- Demonstrates component-based architecture  
- Implements clean state management  
- Showcases glassmorphism UI styling  
- Provides smooth and responsive user interactions  
- Maintains a minimal yet futuristic design approach  

---

## 🚀 Key Features

- ➕ Add New Tasks  
- ✏️ Edit Existing Tasks  
- ❌ Delete Tasks  
- ✅ Mark Tasks as Completed  
- 🔄 Automatic Separation of Active & Completed Tasks  
- 📱 Fully Responsive Layout  
- 🧊 Custom Glass-Style Navbar and Footer  
- 🧩 Clean Component-Based Architecture  

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)  
- Tailwind CSS v4  
- UUID (Unique Task ID generation)  

---

## 🧩 Application Flow

1. User lands on the application interface  
2. User adds a new task using the input field  
3. Tasks are displayed dynamically in the active section  
4. Tasks can be edited, marked as completed, or deleted  
5. Completed tasks are automatically separated  
6. UI updates instantly using React state management

## 📂 Project Structure

The project is organized into reusable components:

- `Navbar`  
- `Footer`  
- `AddTask`  
- `DisplayTask`  
- `EditTask`  
- `CompletedTask`  
- `Todo` (Main Logic Component)  
- `App` (Layout Structure)  
- `main.jsx` (Application Entry Point)
